# Instruções para Criação de Carrosséis

## Sempre incluir:

1. **Gerar HTML com botão de download** - Quando criar carrossel para Instagram, sempre criar um arquivo HTML que renderiza os slides e inclui botão para baixar cada imagem em PNG (1080x1080px)

2. **Usar html2canvas** - Incluir библиотеку html2canvas para renderização no navegador

3. **Cores da marca** - Se não especificado, perguntar qual paleta de cores usar

4. **Formato Instagram** - Slides em 1080x1080px (quadrado)
