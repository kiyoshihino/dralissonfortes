# Skill: Agente de Copywriting

Esta skill cria todo conteúdo textual de marketing na voz e tom da marca do cliente.

## Fluxo de Trabalho

### 1. Configuração Inicial

**Coletar amostras da voz do cliente:**
- E-mails enviados anteriormente
- Posts nas redes sociais
- Site atual
- Materiais de vendas
- Respostas a clientes
- Entrevistas (se necessário)

**Definir parâmetros:**
- Tom de voz: Formal, casual, irreverente, técnico
- Palavras-chave da marca
- Frases de efeito utilizadas
- Palavras proibidas
- Formato de tratamento (tu/você)
- Emoji usage (sim/não)

### 2. Análise da Voz

Criar perfil de copywriting:

```
=== PERFIL DE VOZ ===

Tom: [Ex: Informal, próximo, semformalidades]
Personalidade: [Ex: Divertido mas profissional]
Palavras-chave: [ex: transformação, resultado, prático]
Frases ícone: [ex: "Vamos evoluir?", "Sem enrolação"]
Palavras a evitar: [ex: solução inovadora, sinergia]
Tratamento: [tu ou você]
Emojis: [sim/não - quais usar]
```

### 3. Tipos de Copy

#### **Anúncios (Ads)**

**Google Ads:**
- Headlines (30 chars): Impacto + benefício
- Descrições (90 chars): CTA + urgência
- Estrutura: Problema → Solução → Resultado

**Meta (Facebook/Instagram):**
- Primária: Emocional (15-25 palavras)
- Descrição: Lógica + CTA
- hook nos primeiros 3 segundos

**LinkedIn:**
- Mais formal
- Foco em credibilidade
- Dados e resultados

#### **E-mails**

**E-mail de apresentação:**
- Assunto: Curto + curiosity
- Abertura: Personalização
- Corpo: Problema → Solução → Benefício
- CTA: Ação única clara

**Sequência de e-mails:**
1. Bem-vindo + entrega do lead magnet
2. Storytelling + problema
3. Solução + prova social
4. Urgência/escassez
5. Caso de sucesso
6. Última chance / Follow-up

#### **Legendas (Redes Sociais)**

**Estrutura:**
- Hook (primeira linha): Impacto
- Corpo: Valor + storytelling
- CTA: Encaminhamento
- Hashtags: Relevantes

**Instagram:**
- Linha 1-2: Gancho
- Linhas 3-10: Conteúdo
- CTA: Perto do final
- 3-5 hashtags

**LinkedIn:**
- Mais longo e substantivo
- Estrutura com blocos
- Dados e insights
- CTA profissional

#### **Página de Vendas**

**Estrutura completa:**
1. **Hero**: Headline + subheadline + CTA
2. **Problema**: Despertar a dor
3. **Solução**: Apresentar o produto
4. **Benefícios**: Lista impactante
5. **Prova social**: Depoimentos
6. **Sobre mim/nós**: Credibilidade
7. **Oferta**: Preço + bônus
8. **Garantia**: Remover risco
9. **Urgência**: Tempo limitado
10. **CTA final**: Reiterar chamada

#### **Scripts**

**Vídeo/Reels:**
- Abertura: Gancho em 3 segundos
- Problema: Identificação
- Solução: Apresentação
- CTA: O que fazer

**Sales Script:**
- Abertura: Rapport
- Descoberta: Perguntas-chave
- Apresentação: Problema → Solução
--objections: Respostas prontas
- Fechamento: Compromisso

### 4. Frameworks de Copy

**AIDA:**
- Attention (Atenção)
- Interest (Interesse)
- Desire (Desejo)
- Action (Ação)

**PAS:**
- Problem (Problema)
- Agitation (Agitação)
- Solution (Solução)

**FAB:**
- Features (Características)
- Advantages (Vantagens)
- Benefits (Benefícios)

**The 4Cs:**
- Clear (Claro)
- Concise (Conciso)
- Compelling (Convincente)
- Credible (Crível)

**PSB:**
- Promise (Promessa)
- Picture (Imagem)
- Proof (Prova)

### 5. Gatilhos Psicológicos

Aplicar conforme contexto:

| Gatilho | Quando Usar |
|---------|-------------|
| Urgência | Ofertas por tempo limitado |
| Escassez | Quantidade reduzida |
| Prova social | Depoimentos, números |
| Autoridade | Credenciais, cases |
| Reciprocidade | Conteúdo grátis |
| Especificidade | Números concretos |
| Storytelling | Conexão emocional |
| Medo | Consequências inação |

### 6. Estrutura de Output

Para cada peça de copy, entregar:

```
=== [TIPO DE COPY] ===

[Conteúdo completo]

--- PARÂMETROS USADOS ---
Tom: [tom utilizado]
Comprimento: [X palavras]
CTA: [chamada para ação]
Gatilhos: [gatilhos aplicados]
```

### 7. Revisão

Antes de finalizar, verificar:
- [ ] Voz consistente com a marca
- [ ] CTA claro e presente
- [ ] Formato adequado para o canal
- [ ] Gramática e ortografia
- [ ] Comprimento apropriado
- [ ] Gatilho de ação presente

## Exemplos

### Exemplo E-mail
```
Assunto: [Nome], você está perdendo dinheiro?

Olá [Nome],

Sabe aquele lead que quase comprou, mas escapou?

A maioria das empresas perde 80% desses leads.

O motivo? Falta de follow-up no momento certo.

[CTA]

Abraço,
[Firma]
```

### Exemplo Legenda Instagram
```
3 razões para você NÃO contratar nossa consultoria:

1. Você prefere continuar tentando sozinho
2. Não tem orçamento para investir em crescimento
3. Está confortável com resultados médios

Se você disse "não" para todas, então somos para você.

Comentário "SIM" que eu te explico como funciona 👇
```

## Boas Práticas

- **Escrever para scanner**: Parágrafos curtos, subtítulos
- **Um CTA por peça**: Foco na ação principal
- **Testar variações**: A/B testing de headlines
- **Atualizar constantemente**:based on performance
- **Manter consistência**: Mesma voz em todos os canais
- **Ser útil**: Conteúdo que agrega valor
