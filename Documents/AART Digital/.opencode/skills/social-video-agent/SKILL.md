# Skill: Agente de Vídeo para Redes Sociais

Esta skill cria vídeos otimizados para Instagram, TikTok, Reels e YouTube Shorts.

## Fluxo de Trabalho

### 1. Planejamento

**Definir objetivo do vídeo:**
- awareness (notoriedade)
- Engajamento (comentários, shares)
- Conversão (cliques, vendas)
- Educacional (ensinar algo)

**Conhecer a plataforma:**
| Plataforma | Duração | Formato | Melhor Horário |
|------------|---------|---------|----------------|
| TikTok | 15-60s | 9:16 | 12h-14h / 19h-21h |
| Instagram Reels | 15-90s | 9:16 | 12h / 18h-20h |
| YouTube Shorts | 15-60s | 9:16 | 15h-18h |
| Instagram Feed | 60s | 1:1 ou 4:5 | 11h-13h |

### 2. Roteiro

**Estrutura de Vídeo (15-30 segundos):**

```
=== ROTEIRO ===

0-3s: HOOK (Gancho)
- Pergunta impactante
- Declaração polêmica
- Resultado promise

3-10s: PROBLEMA
- Identificar com a dor
- Mostrar frustração

10-20s: SOLUÇÃO
- Apresentar a solução
- Mostrar benefício

20-30s: CTA
- Ação para espectador
- "Comente X", "Compartilhe"
```

**Tipos de Conteúdo:**

| Formato | Estrutura | Exemplo |
|---------|-----------|---------|
| Tutorial | Problema → Passo a passo → Resultado | "Como fazer X em 30 segundos" |
| Before/After | Antes → Transformação → Depois | "Sua sala antes e depois" |
| List | Gancho → Lista → CTA | "3 erros que você comete no Instagram" |
| Storytime | Mistério → Revelação → Lição | "O que aconteceu quando eu..." |
| Trends | Áudio viral → Adaptar → CTA | Usar trending sounds |
| Product Show | Problema → Demo → Benefício | "Esse produto mudou minha vida" |

### 3. Gravação

**Setup Básico:**
- Smartphone (galaxy S/iPhone recente)
- Ring light
- Tripé ou estabilizador
- Microphone lapela (opcional)

**Enquadramento:**
- selfie (braço esticado ou bastão)
- Profundidade de campo (bokeh)
- Regra dos terços
- Olhar para lente (não tela)

**Iluminação:**
- Luz natural (janela lateral)
- Ring light frontal
- Evitar contraluz

### 4. Edição

**Ferramentas:**
- CapCut (iniciante → avançado)
- InShot (rápido)
- Premiere Rush (profissional)
- DaVinci Resolve (avançado)
- After Effects (motion graphics)

**Estrutura de Edição:**

```
1. Importar clips
2. Cortar/clipar (ritmo rápido)
3. Sincronizar com áudio
4. Adicionar transições
5. Texto sobreposto (captions)
6. Efeitos visuais
7. Correção de cor
8. Exportar
```

**Elementos Essenciais:**

| Elemento | Função | Ferramenta |
|----------|--------|------------|
| Captions | Acessibilidade + mute viewing | CapCut auto-caption |
| Trending Audio | Algoritmo | TikTok sound library |
| Efeitos | Engajamento | Stickers, effects |
| Texto | Ênfase | Text overlay |
| Speed | Ritmo | Speed ramp |

### 5. Otimização

**Para TikTok:**
- Usar 3-5 hashtags (mix popular + niche)
- Descrição com palavras-chave
- Colocar CTA nos primeiros comentários
- Usar trending sounds
- Postar 3-5x/dia (testar)

**Para Instagram Reels:**
- Hashtags relevantes (3-10)
- Criar capa ( thumbnail)
- Title + descrição
- Marcar localização
- Duplicar para Instagram Feed

**Para YouTube Shorts:**
- Title keyword-optimized
- Descrição com timestamp
- Hashtags #Shorts
- Thumbnail customizada

### 6. Especificações Técnicas

| Plataforma | Resolução | Codec | Bitrate | FPS |
|------------|-----------|-------|---------|-----|
| TikTok | 1080x1920 | H.264 | 8-12 Mbps | 30/60 |
| Reels | 1080x1920 | H.264 | 8-12 Mbps | 30/60 |
| Shorts | 1080x1920 | H.264 | 8-12 Mbps | 30/60 |

**Áudio:**
- Taxa de amostragem: 44.1kHz
- Canais: Mono ou Stereo
- Nível: -14 LUFS

### 7. Métricas de Sucesso

| Métrica | Bom | Excelente |
|---------|-----|-----------|
| Taxa de visualização | >50% | >70% |
| Taxa de conclusão | >40% | >60% |
| Engajamento | >5% | >10% |
| Compartilhamentos | >1% | >3% |
| Saves | >2% | >5% |

### 8. Checklist Pré-Publicação

- [ ] Vídeo tem gancho nos primeiros 3 segundos
- [ ] Audio está sincronizado
- [ ] Captions presente e legível
- [ ] Qualidade de vídeo boa (luz, estabilidade)
- [ ] CTA claro
- [ ] Hashtags adequadas
- [ ] Thumbnail atrativa
- [ ] Descrição otimizada
- [ ] Formato correto (9:16)
- [ ] Arquivo exportado corretamente

## Tendências 2026

- Vídeos com IA (avatares, voiceover)
- Hiper-lapse (velocidade 2x-4x)
- Looping videos
- UGC (user-generated content)
- Vídeos respondedores (reply to comment)
- Serial content (série de vídeos)

## Boas Práticas

- **Ritmo rápido**: Cut a cada 2-3 segundos
- **Som original + música**: Balancear
- **Legendas**: 80% assiste sem som
- **Consistência**: Postar regularmente
- **Testar**: A/B test thumbnails e hooks
- **Engajamento**: Responder comentários
- **Analytics**: Acompanhar e otimizar
